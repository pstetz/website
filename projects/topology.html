Introduction

Thinking of space on the very smallest scales is really fun. While a very long time ago people predicted matter might have a smallest form, it was only recently that people discovered energy can only be transferred in packets. Extending these tiny packets to other "quata" has been one of the biggest successes in physics! (By success, physicists are referring to almost purely reproducible predictions by the way. 2.) Fermat's Last Theorem

Another way of thinking about Fermat's Last Theorem is that you can use two N-cubes like Legos to form another N-cube.

This can be done by keeping one of the cubes intact and rearranging the bits of the other to form the final cube. Figure 2 probably expresses this better than any words.

So given an N-cube, we can think of how to add layers to create another, bigger N-cube. Well if we want a shell that is \( d \) thick, then we need a (n-1)-orthotope that is d thick for each possible section. The unique possibilities are removing one axis, so there are N many of them. This by itself adds \( N a^{(N-1)} d \) "units". However (n-1)-othotopes by themselves won't create the shell we want (think of 2-D case, we need a corner piece too), we need to add (n-2) othotopes, (n-3) othotopes, and so on. There will be \( N \choose i \) many of them where \( i \) is the number of axes removed. Including these different dimensional orthotopes we do get a perfect shell and have a new N-cube. Summing up the contributions from the orthotopes gives the series \[ \sum_{i=1}^{N} {N \choose i} a^{(N - i)} d^{i} \]

Therefore assuming a shell is possible to construct, there's the following relation between our starting N-cube, it's shell, and the bigger N-cube. \[ a^N + \sum_{i=1}^{N} {N \choose i} a^{(N -i)} d^{i} = c^N \]

Fermat's Last Theorem says that if \( N \gt 2 \), then there are no integers \( a, d, c \) that solve the above equation

Going back to 2D, the above equations gives \( a^2 + 2 a d + d^2 = c^2\). We can see that when \( a=4 \) and \( d=1 \), we get \( 25 = c^2 \). Since 25 is a perfect root, it means adding one layer to a square with length \( a \), will create another square with length \( c = a + d \)

One way communication

Let's say there's an alien distance "a" away from us. Using Hubble's law we can calculate how far away it is some time, "t", later. \[ a(t) = a + \int^t_0 v \, dt' \rightarrow a + \int^t_0 H_0 d(t') \, dt' \]

Slightly trickier, let's find the formula for an inbound light ray (something traveling at speed "c") that starts distance "d" away. We'll call the light's position "p". The light is moving toward us, but space is pushing the light away from us so we get \[ p(t) = d - \int^t_0 (c - v) \, dt' \rightarrow d - \int^t_0 (c - H_0 p(t')) \, dt' \]

So speaking loosely, the lightrays "speed" is \( \frac{dp}{dt} = c - H_0 p(t) \). This means that if the initial distance is farther than some certain amount, \( p(t=0) = \frac{c}{H_0} \), we will never see the lightray. If it's less then this amount then \( \frac{dp}{dt} \) will be positive and the lightray will approach, albeit slowly.

The above are first order differential equations that we can solve. \[ a(t) = a(0) e^{H_0 t} \] \[ p(t) = (p(0) - \frac{c}{H_0}) e^{H_0 t} + \frac{c}{H_0} \]

So anything past \( d_0 \equiv \frac{c}{H_0} \) will never reach us, if it's between \( d_0 \) and \( d_1 \), it will reach us once, etc. \[ d_n = \frac{d_0}{n + 1} \] \[ t_n = \frac{1}{H_0} ln(\frac{d_0}{d_0 - d_n}) = \frac{1}{H_0} ln(\frac{n + 1}{n}) \]

Interesting properties

Working with these equations are pretty fun and give simple solutions to quantities we might care about. \[ \sum^{n}_{i=1} t_i = \frac{1}{H_0} ln(n + 1) \] \[ t_i - t_j = \frac{1}{H_0} ln( \frac{i (j + 1)}{j (i + 1)}) \]

The equations are fairly robust as well. We can build a clock that measures time by seeing how often light bounces between two mirrors separated a distance apart (much like the one Einstein used in Special Relativity). Somewhat surprisingly, it doesn't matter the distance apart! A clock with mirrors \( d \) apart will match one with mirrors \( \frac{d}{2} \) apart!
